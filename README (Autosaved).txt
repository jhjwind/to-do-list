Some neat features:
1. Message board:
	As soon as any entry is fetched, updated, added or deleted, a notification will pop up on the top right and disappear after two seconds.

2. Delete all:
	Aka massive weapon. This option will be provided when there are more than two entries.

3. Update and Delete:
	These two options will show up when user mouseover any entry. When user chooses to update but cancels operation afterwards, the original entry is restored.

4. Clear button:
	In the text input area, a clear button is provided.
